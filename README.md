## For developers
Please refer to the [Wiki Section][wiki-url].

## Installation
1. Download and setup [Sponge Server](https://spongepowered.org)
2. Put the [latest version][releases-url] of Monument into the `mods/plugins` folder

### Requirements
- Minecraft 1.12.2
- SpongeForge v7.3.0
